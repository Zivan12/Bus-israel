# Bus Israel Map Ready

Готовый проект Node.js + Express с картой Leaflet.

## Что умеет
- показывает автобусы на карте
- фильтр по компании
- фильтр по маршруту
- показывает номер автобуса, компанию и направление
- работает через сервер-прокси

## Запуск локально
npm install
npm start

Открой:
http://localhost:3000
